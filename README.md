# Rencrypt
这是一个为 `JavaScript` 提供的二次加密/解密方案

已经过功能性测试

对文本进行二值化并利用自写算法转换为特定格式16进制文本，支持二进制/16进制/字符串文本

最高支持长度0xffffffff(4294967295)的 `二进制` 文本或长度在536870911以内的 `字符串` (忽略编译器/内存影响)
